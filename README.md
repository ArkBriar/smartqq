Smart QQ C++
================

从[Smart QQ Java](https://github.com/ScienJus/smartqq)改写。

DEPENDENCY
----------------

1\. [CPR](https://github.com/whoshuu/cpr) (LICENSE MIT)
2\. [JSON for Modern C++](https://github.com/nlohmann/json) (LICENSE MIT)
3\. libcurl, version>=7.22

BUILD
----------------

可以通过以下方式在linux上编译测试

> mkdir build
> cd mkdir
> cmake .. & make
